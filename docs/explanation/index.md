# Explanation

Explanation pages help you **understand** how Snakie works and why. They're for
reading when you're curious — no task to finish, just ideas to explore.

- [What is MicroPython?](what-is-micropython.md) — the language your board runs.
- [How Snakie talks to your board](how-snakie-talks-to-your-board.md) — the clever
  trick that lets Snakie run your code and read files over one USB cable.
- [The Board View](board-view.md) — how Snakie reads your code and draws your board.
- [The Parts Library](parts-library.md) — where parts come from and how they're shared.
- [The Robot View](robot-view.md) — how Snakie builds a 3-D robot from parts.
- [Themes and skins](themes.md) — the "skeuomorph" look and light/dark modes.

!!! tip "Ready to do something?"
    Explanation is for understanding. When you want to *make* something, head to
    the [Tutorials](../tutorials/index.md) or [How-to guides](../guides/index.md).
